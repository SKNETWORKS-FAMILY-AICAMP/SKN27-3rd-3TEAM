# 화면설계서 (UI/UX Design Specification)

**프로젝트명:** 포켓몬 AI 챗봇  
**문서 버전:** v1.1  
**작성일:** 2025-05-14  
**최종 수정:** 2025-05-14 (web_search 뱃지 제거, CRAG 상태 표시 추가)

---

## 1. 개요

본 문서는 포켓몬 AI 챗봇의 사용자 인터페이스 구조와 각 화면의 구성 요소, 동작 방식을 정의한다.  
현재 버전은 **웹 기반 채팅 UI**를 기준으로 작성되며, 백엔드 API 완성 후 연동한다.

---

## 2. 화면 목록

| 화면 ID | 화면명 | 설명 |
|---------|--------|------|

## 3. 화면 상세 설계

### SCR-01: 메인 채팅 화면

```
┌─────────────────────────────────────────────────────────────┐
│  [≡ 세션 목록]          🎮 포켓몬 AI 챗봇        [모델 ▾]  │
├──────────────┬──────────────────────────────────────────────┤
│              │                                              │
│  📋 세션     │   💬 대화 영역                               │
│  사이드바    │                                              │
│  (SCR-02)    │   ┌──────────────────────────────────────┐  │
│              │   │ 🤖  안녕하세요! 포켓몬 박사입니다.   │  │
│  ─────────── │   │     무엇이든 물어보세요!             │  │
│  + 새 대화   │   └──────────────────────────────────────┘  │
│  ─────────── │                                              │
│  • 피카츄... │   ┌──────────────────────────────────────┐  │
│  • 리자몽... │   │ 👤  리자몽 공격력 알려줘             │  │
│  • 불꽃타입  │   └──────────────────────────────────────┘  │
│              │                                              │
│              │   ┌──────────────────────────────────────┐  │
│              │   │ 🤖  리자몽의 공격력은 104입니다.     │  │
│              │   │     [🗄 search_pokemon_db]            │  │
│              │   │     | 항목    | 수치 |               │  │
│              │   │     | 공격    | 104  |               │  │
│              │   │     | 특수공격| 109  |               │  │
│              │   └──────────────────────────────────────┘  │
│              │                                              │
│              ├──────────────────────────────────────────────┤
│              │  ┌────────────────────────────────┐  [전송] │
│              │  │ 질문을 입력하세요...            │   ↑    │
│              │  └────────────────────────────────┘        │
└──────────────┴──────────────────────────────────────────────┘
```

**구성 요소:**

| 요소 | 타입 | 설명 |
|------|------|------|
| 헤더 | 고정 바 | 로고, 세션 토글, 모델 선택 드롭다운 |
| 대화 영역 | 스크롤 컨테이너 | 사용자/AI 메시지 버블 렌더링 |
| AI 메시지 버블 | 카드 | 마크다운 렌더링, 툴 뱃지 포함 |
| 사용자 메시지 버블 | 카드 | 우측 정렬, 배경색 구분 |
| 입력창 | textarea | 멀티라인, Enter 전송 / Shift+Enter 줄바꿈 |
| 전송 버튼 | 아이콘 버튼 | 입력 없을 시 비활성화 |

**인터랙션 규칙:**

- 메시지 전송 후 입력창 초기화
- AI 응답 수신 중 스트리밍 텍스트 표시 (타이핑 애니메이션)
- 새 메시지 도착 시 자동 스크롤 다운
- 대화 영역 클릭 시 자동 스크롤 중지

### SCR-02: 세션 사이드바

```
┌──────────────────────┐
│  💬 대화 목록         │
│  ──────────────────  │
│  [+ 새 대화 시작]     │
│  ──────────────────  │
│  📝 피카츄 진화 경로  │ ← 활성 세션 (하이라이트)
│     gpt-4o-mini      │
│     2025-05-14 14:32 │
│  ──────────────────  │
│  📝 불꽃타입 포켓몬   │
│     gpt-4o-mini      │
│     2025-05-13 09:10 │
│  ──────────────────  │
│  📝 리자몽 스탯비교   │
│     gpt-4o-mini      │
│     2025-05-12 21:45 │
│                      │
│  [🗑 삭제] ← 호버 시  │
└──────────────────────┘
```

**동작:**

- 세션 클릭 → 해당 세션의 대화 내역 로드
- `+` 버튼 → 새 세션 생성 및 전환
- 삭제 아이콘 → 확인 모달 후 세션 삭제
- 세션 제목은 첫 번째 사용자 메시지 앞 60자 자동 생성

### SCR-03: 모델 선택 패널

```
┌─────────────────────┐
│  🤖 모델 선택        │
│  ─────────────────  │
│  ● gpt-4o-mini      │ ← 현재 선택
│    빠른 응답, 기본값 │
│  ─────────────────  │
│  ○ gemma4:e2b       │
│    로컬 실행 (Ollama)│
└─────────────────────┘
```

**동작:**

- 드롭다운 형태로 헤더에서 접근
- 모델 변경은 현재 세션에 즉시 적용
- 새 세션 생성 시 선택된 모델이 기본값으로 저장

---

### SCR-04: 툴 사용 뱃지 (Tool Badge)

AI 답변 버블 하단에 사용된 툴을 뱃지 형태로 표시한다.

```
🤖  리자몽의 진화 경로는 다음과 같습니다: ...

    [🗄 search_pokemon_db]  [🔗 search_evolution_chain]
```

| 툴 | 뱃지 색상 | 아이콘 |
|----|---------|--------|
| `search_pokemon_db` | 파란색 | 🗄 |
| `search_flavor_text` | 초록색 | 🔍 |
| `search_evolution_chain` | 보라색 | 🔗 |
| `search_type_relations` | 주황색 | ⚔️ |
| `search_pokemon_weakness` | 빨간색 | 💀 |

> ⚠️ `web_search` 툴은 제거되었습니다. 더 이상 뱃지가 표시되지 않습니다.

**CRAG 재검색 표시:**  
CRAG가 관련성 없음을 판단하여 재검색을 유도한 경우, 뱃지 옆에 🔄 아이콘을 추가하여 재시도가 있었음을 사용자에게 알린다.

```
    [🔍 search_flavor_text 🔄]  [🔍 search_flavor_text]
```

### SCR-05: 에러 알림

```
┌─────────────────────────────────────────────┐
│  ⚠️  Neo4j 연결에 실패했습니다.             │
│      진화/타입 관련 기능이 제한됩니다.       │
└─────────────────────────────────────────────┘
```

- 인라인 노출 (모달 지양)
- 오류 유형별 메시지 구분 (DB 연결 실패 / BM25 빌드 실패 / 시간 초과)
- 5초 후 자동 소멸 또는 X 버튼으로 닫기

## 4. 공통 UI 규칙

### 4.1 타이포그래피

| 요소 | 폰트 크기 | 굵기 |
|------|---------|------|
| 헤더 타이틀 | 18px | Bold |
| 메시지 본문 | 14px | Regular |
| 메타 정보 (시간, 모델명) | 11px | Light |
| 뱃지 레이블 | 11px | Medium |

### 4.2 색상 팔레트

| 용도 | 색상 코드 |
|------|---------|
| Primary (포켓몬 레드) | `#EE1515` |
| AI 메시지 배경 | `#F8F9FA` |
| 사용자 메시지 배경 | `#E3F2FD` |
| 사이드바 배경 | `#1A1A2E` |
| 텍스트 메인 | `#212121` |
| 텍스트 서브 | `#757575` |

### 4.3 반응형 기준

| 브레이크포인트 | 레이아웃 |
|--------------|---------|
| ≥ 1024px | 사이드바 고정 표시 |
| 768px ~ 1023px | 사이드바 오버레이 |
| < 768px | 사이드바 숨김, 햄버거 메뉴 |

## 5. 접근성 (Accessibility)

- 모든 버튼에 `aria-label` 속성 부여
- 키보드 탐색 지원 (Tab 순서 준수)
- 색상 외 아이콘·텍스트로 상태 이중 표현
- 스크린 리더를 위한 AI 응답 완료 알림 (`aria-live`)
