# 🥋 격투 타입 풀어태커(순수 체급) 엔트리 리포트 (MD)

본 문서는 가라르지방의 격투 전문가 **채두(Bea)**의 테마를 바탕으로 구성되었습니다. 변화기를 배제하고 오직 강력한 물리 타격과 기술의 견제폭으로만 승부하는 **'순수 체급형'** 격투 타입 엔트리입니다.
