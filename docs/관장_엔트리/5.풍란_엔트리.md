# 🌪️ 비행 타입 풀어태커(순수 체급) 엔트리 리포트 (MD)

본 문서는 변화기(상태이상, 랭크업, 회복 등)를 일절 배제하고, 오직 **'순수 체급(깡딜)'과 4개의 공격기**만으로 상대를 찍어누르는 상남자/상여자 스타일의 비행 타입 엔트리입니다.

---

## 📋 포켓몬 상세 엔트리 (풀어태커 형태)

### 1. 둥실라이드 (♂/♀)
* **레벨:** 50
* **타입:** 고스트 / 비행
* **특성:** 유폭
* **지닌 도구:** 구애안경 (특수공격 1.5배)
* **기술배치:**
    * 섀도볼
    * 10만볼트
    * 사이코키네시스
    * 에어슬래시
