[상황] 포켓몬 배틀 시뮬레이터를 개발하는 프로젝트에서 그래프 DB 구축 중
[목적] 턴제 배틀 엔진이 판단에 활용할 전투 지식을 그래프로 설계
[설계]
- 노드 후보 대상 및 예시
    - 포켓몬: 망나뇽, 한카리아스, 비버통, 글라이거, 누오, 무우마, 조로아크, 아머까오, 모크나이퍼, 님피아, 껍질몬, 블레이범, 장크로다일, 토오, 파오젠, 미라이돈, 절각참, 홍수몬, 루카리오, 단단지, 가디안, 이어롭 등
    - 타입: 불꽃, 물, 풀, 전기, 드래곤, 얼음, 페어리, 땅, 바위, 강철, 비행, 벌레, 독, 악, 에스퍼, 격투, 노말, 고스트
    - 특성: 부유, 저수, 엽록소, 원격, 프레셔, 날카로운눈, 설경, 가뭄, 잔비, 돌머리, 부풀린가슴, 맹화, 타오르는불꽃, 방탄, 가속, 위협, 긴장감 등
    - 기술(물리 / 특수 / 변화): 인파이트, 공중날기, 구멍파기, 지진, 솔라빔, 떨어뜨리기, 몸통박치기, 파괴광선, 비바라기, 쾌청, 전광석화, 신속, 드래곤크루, 절대영도, 맹독, 독찌르기, 지옥찌르기, 불꽃펀치, 번개펀치, 냉동펀치, 냉동빔, 나이트헤드, 모래뿌리기, 그림자분신, 화염방사, 물대포, 연옥, 불대문자, 배북, 오로라베일, 웅크리기, 단단해지기, 땅가르기, 역린, 용성군, 용의춤, 칼춤, 유턴, 대타출동, 고속스핀, 이판사판태클, 자폭, 파도타기, 분노의앞니, 10만볼트, 번개, 뽐내기, 짓밟기, 하이드로펌프, 덩쿨채찍, 아쿠아제트, 수면가루, 눈싸라기, 천사의키스 등
    - 도구: 
    - 환경: 모래바람, 비, 눈, 쾌청, 매직룸, 원더룸, 중력, 트릭룸, 그래스필드, 미스트필드, 사이코필드, 일렉트릭필드
    - 상태이상: 독, 맹독, 화상, 마비, 잠듦, 얼음, 혼란, 풀죽음, 교체불가, 헤롱헤롱, 소금절이, 멸망의노래, 길동무
- 포켓몬을 노드로 구성하면 갖게 될 속성 후보: 이름, 키, 몸무게, 스탯(HP, 공격, 방어, 스피드, 특수공격, 특수방어)
- 기술을 노드로 구성하면 갖게 될 속성 후보: PP, 위력, 명중률, 분류(물리, 특수, 변화), 접촉여부
- 포켓몬 노드가 관련이 있다고 생각되는 요소
    - 타입: 모든 포켓몬은 1개 또는 2개의 타입을 반드시 갖는다.
    - 특성: 모든 포켓몬은 종마다 1개 또는 2개의 알려진 특성과 1개의 숨겨진 특성 후보가 존재하고 그 중 1개만을 갖는다.
    - 기술: 모든 포켓몬은 종마다 배울 수 있는 기술 후보가 존재한다.
- 타입 노드가 관련이 있다고 생각되는 요소
    - 타입: 타입 간에는 상성 관계가 존재한다.
    - 특성: 특성 중에는 타입에 어떠한 영향을 주는 특성이 존재한다
    - 기술: 모든 기술은 1개의 타입만을 갖는다
- 기술 노드가 관련이 있다고 생각되는 요소
    - 기술: 기술 중에는 포켓몬 자신 혹은 상대방 포켓몬의 다른 기술에게 영향을 주는 어떤 기술이 존재한다.
    - 포켓몬: 기술 중에는 포켓몬 자신 혹은 상대방 포켓몬의 스탯(스피드, 공격, 방어, 특수공격, 특수방어)을 일시적으로 변화시키는 어떤 기술이 존재한다.
    - 특성: 기술 중에는 포켓몬이 보유한 특성에 영향을 주는 어떤 기술이 존재한다.

- 특성 노드와 관련이 있다고 생각되는 요소
    - 포켓몬: 특성 중에는 포켓몬 자신의 스탯(스피드, 공격, 방어, 특수공격, 특수방어)을 영향을 주거나 상대방 포켓몬의 스탯에 영향을 주는 어떤 특성이 존재한다.
    - 기술: 특성 중에는 기술에 영향을 주는 어떤 특성이 존재한다.
    - 특성: 특성 중에는 다른 특성에 영향을 주는 어떤 특성이 존재한다.

- 도구 노드가 관련이 있다고 생각되는 요소
    : 도구 중에는 특정 타입의 기술을 강화시키는 도구, 특정 능력치를 올려주는 도구, 특정 포켓몬만 사용할 수 있는 도구, 특정 특성을 부여/무시/강화하는 도구, 타입과 별도로 특정 기술들을 강화하는 도구가 존재한다.
