# 피피고 (Pipigo) - 귀여운 포켓몬과 함께하는 AI 양방향 번역기

**피피고(Pipigo)**는 당신의 웹 브라우징을 더욱 즐겁게 만들어주는 가상 펫 기반의 지능형 번역 확장 프로그램입니다.

## 주요 기능

- **스마트 양방향 번역**: 언어 자동 감지 기능을 통해 한-영, 영-한 번역을 즉시 수행합니다.
- **가상 펫 인터페이스**: 화면 위를 산책하고 사용자의 시선을 따라가는 4종의 고화질 포켓몬 파트너(피카츄, 야돈, 리자몽, 꼬부기)를 제공합니다.
- **사용자 맞춤형 제어**: 파트너 활성화/비활성화 및 위치 고정 모드를 지원합니다.
- **로컬 리소스 최적화**: 모든 캐릭터 리소스를 로컬에서 관리하여 안정적이고 빠른 실행이 가능합니다.

---

## 개인정보처리방침 (Privacy Policy)

본 확장 프로그램은 사용자의 프라이버시를 최우선으로 생각하며, 구글의 보안 정책을 엄격히 준수합니다.

### 1. 정보 수집 및 이용

- **수집하지 않는 정보**: 본 서비스는 사용자의 브라우징 기록, 개인 신상 정보, 계정 정보 등을 일절 수집하거나 외부 서버로 전송하지 않습니다.
- **번역 데이터**: 사용자가 번역을 위해 입력한 텍스트는 오직 번역 수행을 위해 Groq AI API 서버로만 전송되며, 어떠한 장치나 서버에도 저장되지 않고 즉시 폐기됩니다.

### 2. 권한 사용 목적 (Permissions)

- **Storage**: 사용자가 선택한 파트너 종류, 위치 고정 상태 등의 설정을 사용자의 로컬 브라우저에만 저장하기 위해 사용합니다. (외부 전송 없음)
- **Host Permissions**: `api.groq.com` 주소는 오직 AI 번역 엔진과의 통신을 위해서만 제한적으로 사용됩니다.

### 3. 데이터 보안

- 모든 실행 코드는 확장 프로그램 패키지 내부에 포함된 로컬 스크립트만 사용하며, 원격 코드 실행(Remote Code Execution)을 절대 수행하지 않습니다.

## 설치 방법

1. 본 저장소를 다운로드하거나 압축을 풉니다.
2. 크롬 브라우저에서 `chrome://extensions`에 접속합니다.
3. 우측 상단의 '개발자 모드'를 활성화합니다.
4. '압축해제된 확장 프로그램을 로드합니다' 버튼을 눌러 `pokemon_papago` 폴더를 선택합니다.
